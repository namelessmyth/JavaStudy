# 项目说明
- 本项目是学习马士兵教育的MCA课程时所建的项目。主要收录学习期间的所有代码。
- 部分代码是直接来源于马士兵（com.mashibing），部分代码为本人编写（com.sjj）
- 项目为多模块项目，在根项目中的为公共依赖。每个模块还有各自的依赖
